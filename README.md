# Sentinel Core - TikTok Landing Page

Landing page for TikTok ad campaigns with newsletter signup and conversion tracking.

## Features

- Newsletter signup form
- TikTok Pixel integration
- Conversion tracking
- Mobile-responsive design

## Setup

1. Set secrets in Streamlit Cloud:
   - `TIKTOK_USERNAME`: Your TikTok username
   - `TIKTOK_PIXEL_ID`: Your TikTok Pixel ID

## Deployment

Deployed on Streamlit Cloud.
